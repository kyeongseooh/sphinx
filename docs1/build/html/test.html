test.s3Recursive(s3_path: list, path_condition: str, recursive: bool)¶

검색 조건에 맞는 s3 path list를 찾아주는 함수

매개변수

  • s3_path (list) – s3_path의 리스트

  • path_condition (str) – 검색 조건

  • recursive (bool) – path를 재귀적으로 탐색 (default=`False`)

how to use
>>> s3Recursive(s3_path, "/*", True)
